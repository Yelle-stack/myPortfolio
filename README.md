# 🖥️ Portfolio – Jelena | Front-End Developer

Ce projet est mon portfolio personnel, conçu pour présenter mes compétences, mes projets web et mes services en tant que développeuse Front-End.
Il met en avant mon parcours, mes réalisations ainsi que plusieurs projets réalisés avec des technologies modernes.

![image alt](https://github.com/Yelle-stack/myPortfolio/blob/12a72f56b529e49c768d40767f3be28f52cf8e32/potfolioimg.png)
## 🚀 Fonctionnalités principales
Section Home : présentation professionnelle, photo, CTA.

Section About : introduction, compétences front-end, liens vers GitHub et LinkedIn.

Section Services : ce que je propose (Figma, Web Design, SEO, VS Code workflow).

Section Portfolio : grille de projets, images, effets hover, liens directs vers GitHub.

Section Contact : formulaire connecté à Web3Forms pour recevoir des messages.

Header responsive : menu mobile avec animation.

Animations élégantes grâce à ScrollReveal.

Design fully responsive (mobile-first) pour tous les écrans.

## 🛠️ Technologies utilisées
HTML5

CSS3 (Grid, Responsive, Animations)

JavaScript

ScrollReveal.js

BoxIcons

Web3Forms API (envoi du formulaire de contact)

## 🎨 Design
Interface moderne et épurée

Palette professionnelle

Animations fluides

Maquettes créées sur Figma

## 📬 Contact intégré
Le formulaire utilise Web3Forms pour envoyer les messages sans backend.

## 📁 Objectif du projet
Créer un portfolio professionnel, clair, élégant et responsive permettant de mettre en valeur mes compétences et mes projets en tant que développeuse front-end.
